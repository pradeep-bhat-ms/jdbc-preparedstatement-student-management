# JDBC PreparedStatement Student Management System

## Project Description

This is a simple JDBC-based Student Management System developed using Java and MySQL. The project demonstrates CRUD (Create, Read, Update, Delete) operations using the JDBC PreparedStatement interface.

---

## Features

* Create Database
* Create Student Table
* Insert Student Records
* Update Student Details
* Delete Student Records
* Fetch Student Records

---

## Technologies Used

* Java
* JDBC

* Eclipse IDE
